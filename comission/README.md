# CoMission 双人任务协作应用

## [😍] 应用简介

CoMission 是一款专为双人协作设计的任务管理应用，通过积分激励和实时通信，让两人之间的任务协作更加高效有趣。

## [✨] 主要功能

### 🎯 任务系统
- **任务分类**: 学习 📚 / 工作 💼 / 生活 🏠
- **积分奖励**: 1-100 积分可配置
- **任务流程**: 发布 → 接受 → 完成 → 审核 → 批准
- **实时同步**: 任务状态变更即时通知

### 🛒 积分商城
- **商品管理**: 创建/编辑/删除商品
- **兑换流程**: 确认弹窗 → 自动下架 → 订单生成
- **订单管理**: 我买到的 / 我卖出的
- **物流跟踪**: 待发货 → 已发货 → 已收货

### 👥 组队系统
- **配对码**: 6 位字母数字组合
- **实时状态**: 队友在线/离线显示
- **积分清零**: 退出组队时双方积分重置

### 🔔 通知中心
- **待发货通知**: 商品被兑换后提醒卖家
- **任务审核**: 任务完成后提醒发布者
- **确认收货**: 卖家发货后提醒买家

### 📊 积分记录
- **收支明细**: 获得/消费完整记录
- **统计卡片**: 累计获得 / 累计消费

## [📦] 安装方法

1. 在飞牛系统应用中心搜索"CoMission"安装
2. 或下载本目录的 `.fpk` 文件到飞牛系统手动安装
3. 或使用 Docker 镜像运行：
   ```bash
   docker run -d -p 3000:3000 --name comission ghcr.io/boater-man/comission:latest
   ```

## [🎮] 使用指南

### 开始使用
1. 输入用户名创建账号
2. 点击"创建组队"生成配对码
3. 将配对码分享给队友
4. 队友输入配对码加入

### 发布任务
1. 进入"任务"页面
2. 点击"+ 发布任务"
3. 填写任务信息（标题/描述/类型/积分）
4. 点击"发布任务"

### 兑换商品
1. 进入"商城"页面
2. 浏览商品并点击"兑换"
3. 确认兑换信息
4. 卖家在"订单管理"发货
5. 买家确认收货完成交易

## [🔋] 技术栈

- **前端**: React 18 + TypeScript + Vite
- **后端**: Node.js + Express
- **实时通信**: Socket.IO
- **数据库**: SQLite
- **样式**: Tailwind CSS
