# PixelBeans 拼豆底稿生成器

## [😍] 应用简介

PixelBeans 是一款功能强大的拼豆（Perler Beads）底稿生成器，专为拼豆爱好者设计。它支持将任意图片转换为像素风格的拼豆图纸，并提供智能颜色映射、区域合并、背景移除等专业功能。

## [✨] 主要功能

- **图片上传与像素化**：支持 JPG/PNG 格式，可调节像素粒度
- **多色板支持**：内置 168 色、144 色、96 色等多种拼豆色板（MARD、COCO、漫漫等）
- **智能颜色合并**：自动合并相似颜色，减少杂色
- **背景移除**：智能识别并移除背景区域
- **颜色排除与重映射**：可排除不需要的颜色并自动重映射
- **实时预览**：悬停查看色号，支持亮色/暗色模式
- **图纸导出**：下载带色号标注的网格图纸
- **统计图导出**：生成颜色统计图，显示每种颜色的使用数量
- **PWA 支持**：可安装为桌面应用，离线使用

## [📦] 安装方法

1. 添加该仓库到FNDEPOT三方应用市场"PixelBeans"安装
2. 下载本目录的 `.fpk` 文件到飞牛系统手动安装
3. 使用 Docker 镜像运行：
   ```bash
   docker run -d -p 3000:3000 --name pixelbeans ghcr.io/boater-man/pixelbeans:latest
   ```

## [🎨] 使用指南

1. **上传图片**：点击上传区域或拖放图片
2. **调整粒度**：滑动粒度滑块控制像素化程度
3. **选择色板**：选择适合的拼豆色板
4. **调整阈值**：调节颜色合并阈值平滑色块
5. **手动编辑**：点击颜色列表排除不需要的颜色
6. **下载图纸**：点击"下载图纸"获取带色号的网格图
7. **导出统计**：下载颜色统计图查看采购清单

## [🔋] 技术栈

- **框架**：Next.js 15 + React 19 + TypeScript
- **样式**：Tailwind CSS 4
- **核心**：Canvas API 用于图像处理
- **部署**：Docker 容器化部署
